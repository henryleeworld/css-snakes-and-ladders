# CSS 蛇梯棋

蛇梯棋是源自於印度的遊戲，有 100 個格子，每個格子都有順序編號，棋盤上繪有蛇與梯子圖案，搭配一個骰子，玩家鄒到第 100 格中就獲勝。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
